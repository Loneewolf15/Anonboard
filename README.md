<h1>👻 Anonboard</h1>
<p>The anonymous social board for radically honest feedback and confessions.</p>
</div>

## Features

- **Anonymous Sending:** Anyone can drop you a message with zero traces.
- **Vanity URLs:** Simple `/:username` structure so you can easily share your board link.
- **Categorized Drops:** Feedback, Compliments, Questions, or General messages.
- **Instant Setup:** Board owners use seamless anonymous authentication to create a board instantly.

## Tech Stack

- React 19 + TypeScript
- Vite + React Router DOM v7
- Tailwind CSS v4 + Framer Motion
- Firebase (Auth, Firestore)

## Quick Start

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Configure Firebase: Set your environment variables or make sure `firebase-applet-config.json` is correctly pointing to your project.
3. Run the application locally:
   ```bash
   npm run dev
   ```

Enjoy the mystery!
